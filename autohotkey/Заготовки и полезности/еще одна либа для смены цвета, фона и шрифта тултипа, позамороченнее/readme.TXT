Основные настройки:

options:

ShowNow (или now) — true или false, показывать или не показывать ToolTip при создании экземпляра объекта. Если параметр не указан, ToolTip будет показан сразу же.
Кароче просто ставь всегда false. Пример: ShowNow: false

FontName (или font) - название шрифта, указывается в ковычках, пример: FontName: "Comic Sans MS"

FontSize (или size) - размер шрифта, пример: FontSize: 20

FontStyle (или style) — стиль шрифта, т.е. bold, italic, underline, strikeout в любом сочетании через пробел.

TimeOut (или time) — время в милисекундах, через которое ToolTip будет скрыт, если число положительное, либо уничтожен, если отрицательное. Можно просто опустить.

BackColor (или back) — цвет фона, пример: BackColor: 0xFFA500

TextColor (или color) — цвет текста, пример: BackColor: 0xFFA500
